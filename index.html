Google Sheets Sync
Sync privately to your own Google Sheet

Changes sync to Sheets automatically when a URL is saved.

  1. Go to sheets.google.com → create a new blank spreadsheet → name it Budget App
  2. Click Extensions → Apps Script
  3. Delete any existing code and paste the full contents of apps-script.gs (included in your ZIP)
  4. Click Deploy → New deployment
  5. Set type: Web app
  6. Set Execute as: Me and Who has access: Only myself
  7. Click Deploy, authorise when prompted, then copy the Web App URL
  8. Paste the URL in the field above and tap Save URL
